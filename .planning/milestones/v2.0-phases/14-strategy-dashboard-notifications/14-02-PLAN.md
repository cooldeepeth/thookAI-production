**Create `frontend/src/hooks/useStrategyFeed.js`:**

Custom hook that manages strategy card state and API calls. Follow the `useNotifications.js` structure exactly.

```
const BACKEND_URL = process.env.REACT_APP_BACKEND_URL;
```

State: `activeCards` (array), `historyCards` (array), `loading` (boolean).

Functions:
- `getAuthHeaders()` -- `useCallback`, reads `localStorage.getItem("thook_token")`, returns `{ Authorization: "Bearer {token}" }` or `{}`
- `fetchActiveCards()` -- `useCallback`, GET `${BACKEND_URL}/api/strategy?status=pending_approval&limit=3`, sets `activeCards` from `data.cards`
- `fetchHistoryCards()` -- `useCallback`, GET `${BACKEND_URL}/api/strategy?status=dismissed&limit=20`, sets `historyCards` from `data.cards`. Also fetch approved cards: GET `${BACKEND_URL}/api/strategy?status=approved&limit=20`, merge both arrays sorted by `created_at` descending
- `approveCard(recommendationId)` -- `useCallback`, POST `${BACKEND_URL}/api/strategy/${recommendationId}/approve`. After success, call `fetchActiveCards()`. Return `data.generate_payload` to caller
- `dismissCard(recommendationId, reason)` -- `useCallback`, POST `${BACKEND_URL}/api/strategy/${recommendationId}/dismiss` with JSON body `{ reason }` if reason provided. After success, call both `fetchActiveCards()` and `fetchHistoryCards()`. Return full response (caller checks `needs_calibration_prompt`)

Init: `useEffect` on mount calls `fetchActiveCards()` and `fetchHistoryCards()` in parallel via `Promise.all`.

Return: `{ activeCards, historyCards, loading, approveCard, dismissCard, refresh: fetchActiveCards }`

All fetch calls include `{ credentials: "include", headers: getAuthHeaders() }`.

**handleApprove function (DASH-02):**
```javascript
const handleApprove = async (card) => {
  setLoadingCardId(card.recommendation_id);
  try {
    const generatePayload = await approveCard(card.recommendation_id);
    // Validate payload has required fields before firing content create
    if (!generatePayload?.platform || !generatePayload?.content_type || !generatePayload?.raw_input) {
      throw new Error("Invalid generate payload -- missing required fields");
    }
    const token = localStorage.getItem("thook_token");
    const res = await fetch(`${BACKEND_URL}/api/content/create`, {
      method: "POST",
      headers: { "Content-Type": "application/json", Authorization: `Bearer ${token}` },
      credentials: "include",
      body: JSON.stringify(generatePayload),
    });
    if (!res.ok) {
      const err = await res.json().catch(() => ({}));
      throw new Error(err.detail || "Content generation failed");
    }
    const data = await res.json();
    navigate(`/dashboard/studio?job=${data.job_id}`);
  } catch (e) {
    toast({ title: "Approval failed", description: e.message, variant: "destructive" });
  } finally {
    setLoadingCardId(null);
  }
};
```

**handleDismiss function (DASH-03):**
```javascript
const handleDismiss = async (card) => {
  setLoadingCardId(card.recommendation_id);
  try {
    const result = await dismissCard(card.recommendation_id);
    if (result?.needs_calibration_prompt) {
      toast({
        title: "Strategy feed recalibrating",
        description: "You've dismissed several recommendations. Your strategy feed will be recalibrated to better match your preferences.",
      });
    }
  } catch (e) {
    toast({ title: "Dismiss failed", description: e.message, variant: "destructive" });
  } finally {
    setLoadingCardId(null);
  }
};
```
