    """Compute HMAC-SHA256 signature for a payload."""
    return hmac.new(
        secret.encode("utf-8"),
        payload_bytes,
        hashlib.sha256,
    ).hexdigest()

    **Step 2: Create backend/routes/n8n_bridge.py**

    New file with:
    - `router = APIRouter(prefix="/n8n", tags=["n8n"])`
    - `_verify_n8n_signature(body_bytes: bytes, signature_header: str) -> bool` — uses `hmac.new(settings.n8n.webhook_secret.encode("utf-8"), body_bytes, hashlib.sha256).hexdigest()` and `hmac.compare_digest()` for constant-time comparison. Returns False if `settings.n8n.webhook_secret` is empty.
    - `POST /callback` — reads raw body bytes, gets `X-ThookAI-Signature` header, verifies signature, parses JSON payload, extracts `workflow_type` field, returns `{"status": "accepted"}`. Raises HTTPException(401) on invalid signature.
    - `POST /trigger/{workflow_name}` — protected by `get_current_user` dependency. Maps workflow_name to n8n webhook URL using a dict: `{"process-scheduled-posts": settings.n8n.workflow_scheduled_posts, "reset-daily-limits": settings.n8n.workflow_reset_daily_limits, ...}` for all 7 workflows. If workflow_name not in dict or value is None, returns 404. Otherwise fires httpx.AsyncClient POST to `f"{settings.n8n.n8n_url}/webhook/{workflow_id}"` with the payload JSON, returns `{"status": "triggered", "workflow": workflow_name}`. On httpx error, returns 502.

    Import pattern: `from config import settings`, `from auth_utils import get_current_user`, `import httpx`, `import hmac`, `import hashlib`, `import json`, `import logging`.
