**Step 2 — Convert Dashboard sub-pages to React.lazy:**

Read `frontend/src/pages/Dashboard/index.jsx` in full.

Find ALL static import statements at the top of the file that import sub-page components (DashboardHome, ContentStudio, Analytics, StrategyDashboard, Settings, etc. — every sub-page import).

Replace ALL of them with `React.lazy()` equivalents. Pattern:
```jsx
// BEFORE (static):
import DashboardHome from "./DashboardHome";
import ContentStudio from "./ContentStudio";
// ... all other sub-page imports

// AFTER (lazy):
const DashboardHome = lazy(() => import("./DashboardHome"));
const ContentStudio = lazy(() => import("./ContentStudio"));
// ... same pattern for every sub-page
```

At the top of the file, change `import React from "react"` (or `import { ... } from "react"`) to include `lazy` and `Suspense`:
```jsx
import { lazy, Suspense } from "react";
```
(If the file uses `React.useState` etc., keep the `React` default import too: `import React, { lazy, Suspense } from "react"`)

**CRITICAL — Suspense placement (per research Pitfall 3):** The `<Suspense>` fallback MUST be placed INSIDE the layout shell, wrapping only the content area — NOT the entire Dashboard component. The sidebar and topbar must never be inside Suspense.

Find the JSX structure in Dashboard/index.jsx. Locate the content area div (typically something like `<div className="flex-1 md:ml-64 ...">` or `<main ...>`). Wrap only the `<Routes>` element (and children) in Suspense:

```jsx
<Suspense fallback={
  <div className="flex items-center justify-center min-h-[60vh]">
    <div className="w-8 h-8 border-2 border-lime border-t-transparent rounded-full animate-spin" />
  </div>
}>
  <Routes>
    <Route path="/" element={<DashboardHome />} />
    {/* ... all routes */}
  </Routes>
</Suspense>
```

Do NOT wrap the sidebar, topbar, or the outer Dashboard container in Suspense.

Preserve ALL route definitions exactly. Only convert the import style and add Suspense around Routes.
