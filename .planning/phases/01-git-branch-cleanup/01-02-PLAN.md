    main has never been updated from dev (it is 59 commits behind). This is a simple fast-forward.

    Step 1 — Checkout main:
      git checkout main

    Step 2 — Fast-forward main to dev:
      git merge dev --ff-only

    The --ff-only flag is a safety net: it will REFUSE to create a merge commit if main has
    diverged. If it fails, STOP and report.

    Step 3 — Verify main now matches dev:
      git diff main dev
    Expected: no output (identical)
      git log --oneline main | head -3
    Expected: same commits as dev tip

    Step 4 — Push main to remote:
      git push origin main

    Step 5 — Switch back to dev as the working branch:
      git checkout dev

    IMPORTANT: CLAUDE.md absolute rule is "NEVER commit directly to main" — this task does NOT
    commit to main. It only fast-forwards the pointer. No new commits are made on main.
    The rule is about not branching work from main, not about fast-forward pointer updates.
  </action>
