From frontend/tailwind.config.js (verified):
  bg-surface       = #0F0F10   (card backgrounds)
  bg-surface-2     = #18181B   (input backgrounds, elevated surfaces)

WRONG vs CORRECT (hard rules):
  bg-[#18181B]           → bg-surface-2
  bg-[#27272A]           → bg-border-subtle
  bg-[#0A0A0B]           → bg-surface
  bg-[#0A0A0A]           → bg-surface
  text-lime-400          → text-lime          (lime-400 is NOT defined — silently no-ops)
  text-lime-500          → text-lime
  from-lime-500/20       → from-lime/20
  via-lime-400/10        → via-lime/10
  to-lime-500/20         → to-lime/20
  text-[#B8E600]         → hover:text-lime/80 (or remove if .btn-primary handles hover)
  style={{ backgroundColor: '#D4FF00' }} → className="bg-lime" (when static)
  style={{ backgroundColor: '#18181B' }} → className="bg-surface-2"
