From frontend/src/lib/api.js (after Plan 01):
```javascript
/**
 * @param {string} path - API path (e.g., "/api/content/create")
 * @param {object} options - fetch options + optional timeout override
 * @returns {Promise<Response>} - raw Response object
 */
export async function apiFetch(path, options = {})
```
- credentials: 'include' is automatic
- CSRF token header injected on POST/PUT/PATCH/DELETE
- Content-Type: application/json set unless body is FormData
- 401 auto-redirects to /auth
- 403 shows permission toast
- 5xx retries once then shows error toast


Target pattern:
