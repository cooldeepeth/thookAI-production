       - Mock GET /render/{id}/status returning "queued" then "rendering" then "done" with url

    4. `@pytest.mark.skipif(not os.environ.get("REMOTION_URL"), reason="Requires running Remotion service")

       - Each request renders a StaticImageCard with minimal inputProps (text="Test {i}", width=600, height=400)
