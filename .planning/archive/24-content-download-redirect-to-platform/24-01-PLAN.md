Platform URL contracts (official):
- LinkedIn: https://www.linkedin.com/shareArticle?mini=true&summary={encoded}
- X/Twitter: https://twitter.com/intent/tweet?text={encoded}  (280 char max — encode the first 280 chars)
- Instagram: No compose URL — show tooltip with copy-paste instructions

```javascript
import JSZip from 'jszip';
const zip = new JSZip();
// for each image URL: fetch → arrayBuffer → zip.file(name, buffer)
const blob = await zip.generateAsync({ type: 'blob' });
// trigger download via anchor click
```

  <name>Task 1: Install JSZip + create contentExport.js utility module</name>
  <files>frontend/package.json, frontend/src/lib/contentExport.js</files>
  <action>
    Install JSZip in the frontend package:
    ```
    cd frontend && npm install jszip
    ```
    This adds jszip to package.json dependencies. No other packages needed — all other functionality (Blob, URL.createObjectURL, anchor click) is native browser API.

    Create `frontend/src/lib/contentExport.js` with four exported pure functions:

    **downloadTextFile(text, platform, date)**
    - Creates a Blob with type "text/plain;charset=utf-8"
    - Filename: `{platform}-{date}.txt` where date is formatted as YYYY-MM-DD (pass `new Date()` as date, format internally)
    - Triggers download: create `<a>` element, set href to `URL.createObjectURL(blob)`, set download attribute, click it, then `URL.revokeObjectURL` to clean up
    - Returns void

    **buildLinkedInUrl(text)**
    - Official LinkedIn share endpoint: `https://www.linkedin.com/shareArticle`
    - Params: `mini=true`, `summary=encodeURIComponent(text)`
    - Returns full URL string

    **buildXUrl(text)**
    - Official X/Twitter tweet intent: `https://twitter.com/intent/tweet`
    - X hard limit: 280 chars. Truncate text to 280 chars before encoding. If text > 280 chars, slice to 277 and append "..." to signal truncation.
    - Params: `text=encodeURIComponent(truncatedText)`
    - Returns full URL string

    **downloadImages(mediaAssets, carousel, jobId)**
    - This is async.
    - Carousel detection: `carousel?.generated && carousel?.slides?.length > 0`
    - If NOT carousel: single image. mediaAssets[0].image_url is the URL. Trigger download via anchor with `download` attribute. Return void.
    - If carousel: fetch each slide image URL, build a zip.
      - Import JSZip from 'jszip'
      - `const zip = new JSZip()`
      - For each slide in carousel.slides (expected shape: `{ image_url: string, slide_number: number }`), fetch the image URL, get `arrayBuffer()`, add to zip as `slide-{N}.jpg` (or .png — check URL extension, default to .jpg)
      - `const blob = await zip.generateAsync({ type: 'blob' })`
      - Trigger download: anchor with href = `URL.createObjectURL(blob)`, download = `carousel-${jobId}.zip`, click, revoke
      - Wrap in try/catch — on error, log to console.error (never silent swallow), rethrow so ExportActionsBar can show error state
    - Returns void

    No default export. Named exports only.
  </action>
