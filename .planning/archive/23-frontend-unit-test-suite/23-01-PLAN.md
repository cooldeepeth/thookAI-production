CRA/CRACO Jest integration: Jest config goes inside `craco.config.js` under a `jest` key.
Example:
```js
module.exports = {
  webpack: { ... },
  jest: {
    configure: {
      moduleNameMapper: {
        '^@/(.*)$': '<rootDir>/src/$1',
      },
      setupFilesAfterFramework: ['<rootDir>/src/setupTests.js'],
    },
  },
};
```
CRA default `setupFilesAfterFramework` points to `src/setupTests.js` (auto-detected if the file exists with the correct name).
