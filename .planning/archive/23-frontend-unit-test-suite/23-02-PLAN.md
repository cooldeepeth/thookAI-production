From frontend/src/lib/api.js — apiFetch behaviors to test:
- Sends credentials: 'include' on every request
- Injects X-CSRF-Token header from csrf_token cookie on POST/PUT/PATCH/DELETE
- Enforces 15s timeout via AbortController; can be overridden via options.timeout
- If caller provides options.signal, no timeout is applied
- Retries once after 1s backoff on 5xx; total attempts = 2
- 401 response: calls window.location.href = '/auth?expired=1' then throws
- 403 response: calls toast({ title: 'Permission denied', ... }) then throws
- 5xx after retry: calls toast({ title: 'Server error', ... }) then returns the response
- Returns raw Response object (callers call .json())

From frontend/src/context/AuthContext.jsx:
- AuthProvider: initializes user=null, loading=true
- On mount: reads ?token param from URL → calls /api/auth/me with Bearer header (Google OAuth path)
- On mount without token: calls checkAuth() → GET /api/auth/me; sets user from response
- checkAuth: on non-ok response → setUser(null)
- login(userData): setUser(userData)
- logout(): POST /api/auth/logout, then setUser(null)
- useAuth(): throws if used outside AuthProvider

From frontend/src/hooks/useStrategyFeed.js:
- On mount: fetches /api/strategy?status=pending_approval&limit=3 and two history endpoints
- approveCard(id): POST /api/strategy/:id/approve → refreshes active cards → returns generate_payload
- dismissCard(id, reason): POST /api/strategy/:id/dismiss with body → refreshes both lists

From frontend/src/hooks/useNotifications.js:
- On mount: fetches /api/notifications and /api/notifications/count
- markRead(id): POST /api/notifications/:id/read
- markAllRead(): POST /api/notifications/read-all
